# Video Assets

## Hero Video
`hero-reel.mp4` — fullscreen hero background video
- Format: MP4 (H.264)
- Duration: 15-30 seconds loop
- Resolution: 1920x1080 minimum
- No audio
- File size: under 10MB recommended

## Hero Poster
`hero-poster.jpg` — fallback image when video cannot play
- Format: JPG
- Resolution: 1920x1080
- Shows first frame or representative still
