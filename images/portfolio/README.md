# Portfolio Image Assets

## Naming Convention
Each project uses: `{slug}-{variant}.jpg`
- `{slug}-cover.jpg` — main cover image (used in grid + detail hero)
- `{slug}-01.jpg` — gallery image 1
- `{slug}-02.jpg` — gallery image 2
- `{slug}-03.jpg` — gallery image 3 (optional)

## Specifications
- Format: JPG (quality 85-90)
- Color space: sRGB
- Resolution: 2400px on the long edge minimum
- Aspect ratios: match the project's `aspectRatio` frontmatter field

## Categories
- street/ — street photography
- travel/ — travel & landscape
- nature/ — nature & wildlife
- portrait/ — portrait & people
- pet/ — pet & animal
- cinematic/ — cinematic & film stills
